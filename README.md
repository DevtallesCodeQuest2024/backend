## Description

This is the backend project.

## Requirements

- Node.js 18.17.1
- Express 4.x
- TypeScript 4.x

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run dev

# build
$ npm run build

# production mode
$ npm run start
```
